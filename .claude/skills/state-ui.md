# Skill: State Management & UI

## Global State (Zustand)
- Stores: `cart`, `address`, `ui`.
- Persistence: Cart state persists in `localStorage`.
- Path: `src/store/`.

## Styling & Components
- Framework: Tailwind CSS.
- Assets: Cloudinary for product images.
- Fonts: `src/config/fonts.ts`.
- Component Pattern: Organized by domain, re-exported in `src/components/index.ts`.
